# ADR-002 - HTML Report

## Decision

The tool will generate a self-contained HTML report with embedded CSS and JavaScript.

## Goals

- Improve readability compared with raw CSV.
- Organize results by server.
- Provide collapsible cards, role badges, filters and search.
- Keep the CSV as the technical source for Excel analysis.

## Constraints

- No internet dependency.
- No external JavaScript or CSS framework.
- The report must be portable as a single file.
